This is a thesis style for the University of Aberdeen School of
Engineering. It is a double-sided "book" formatting (yes, this is
allowed) with nomenclature, and fancy chapter formatting.

This folder contains several files. The most important are listed
below.

thesis.tex : The file which you should edit to fill out your thesis,
this file currently contains some examples on the usage of the
template and LaTeX.

example.pdf : Sample output of the style, built directly from
thesis.tex.

thesis.bib : Your bibliography file, where all of your references
should be stored.

You will need to makelist to build the nomenclature, more information
is given in the thesis.tex file!
